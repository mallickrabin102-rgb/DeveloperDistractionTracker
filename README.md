# Developer Distraction Tracker

This is a simple Java console project that tracks coding sessions and distractions.
The program helps a developer see how much time they spend coding and how much time is wasted on distractions.

## Features

* Add coding sessions
* Add distraction sessions
* View all sessions
* Calculate focus score

## Concepts Used

* Java OOP
* Inheritance
* Polymorphism
* Arrays
* Scanner class

## How to Run

Compile the program:

javac DeveloperDistractionTracker.java

Run the program:

java DeveloperDistractionTracker
